Урок №6. Классы, объекты и методы.


В этом уроке вы познакомитесь с термином класс, объект и метод, также рассмотрим примеры создания объектов и их взаимодействие.

Тест 1


В каком случае объект создаётся неверно:

  1. Cat cat = new Cat();
  2. Dog dog = new Dog;
  3. Bird cat = new Bird();
  4. Cat cat;
    cat = new Cat();
  5. dog = new Dog();

Тест 2


Какие утверждения не верны:

  1. Класс это описание объекта и методов работы с ним.
  2. Экземпляр класса это описание объекта.
  3. Объект это экземпляр класса.
  4. Статические методы можно вызвать без создания объекта.
  5. Нестатические методы можно вызвать без создания объекта.

Задача 1

Написать класс Bird. Написать метод fly.
При вызове метода fly - на консоли должно печататься: "I can fly"

Задача 2

Поменяйте сигнатуру метода fly так, чтобы он принимал значение
При вызове метода fly - на консоли должно печататься: "I can fly" + значение, переданное методу.

Задача 3

Создайте метод flyBack который должен возвращать значение, которое ему передали, умноженное на 2 и выведите результат на консоль.

Ответы:

Тест 1

Вариант 2 не верен - не указаны круглые скобки.
Вариант 5 не верен - не объявлен тип переменной.
Вариант 3 ВЕРЕН - название переменной мы можем присвоить любое, но логичней назвать переменную cat.

Тест 2

Номер 2 не верен - экземпляр класса и есть сам объект, а не его описание.
Номер 5 не верен - для вызова нестатических методов нужно создавать экземпляр класса (объект).


Задачи 1-3

public class Bird {
    public void fly(int distance) {
        System.out.println("I can fly " + distance);
    }

    public int flyBack(int distance) {
        return distance * 2;
    }

    public static void main(String[] args) {
        Bird bird = new Bird();
        bird.fly(100);
        System.out.println(bird.flyBack(100));
    }
}
