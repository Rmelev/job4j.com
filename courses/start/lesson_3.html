Урок №3. Первая программа.


В этом уроке мы создадим нашу первую программу Hello World, а таккже разберёмся чем отличается запуск от компиляции.

Тест 1


Выберите неверные утверждения:

  1. Тело класса или метода ограничивается фигурными скобками
  2. Тело класса или метода ограничивается круглыми скобками
  3. Метод должен находиться внутри тела класса
  4. Метод не должен находиться внутри тела класса

Тест 2


Какие определения не относятся к компиляции:

  1. Компиляция простыми словами это запуск программа
  2. Компиляция простыми словами это сборка программы
  3. Компилятор Java преобразовывает байт-код в исходный код
  4. Компилятор Java преобразовывает исходный код в байт-код

Задача 1


По аналогии с видео написать программу, которая будет выводить сообщение в консоль:
Hello Job4j!

Задача 2


Написать программу, которая будет выводить в консоль:
Hello Job4j! каждое слово с отдельной строки:
Hello
Job4j!

Ответы:


Тест 1

Утверждения под номерами 2 и 4 - не верны.


Тест 2

Утверждения под номерами 1 и 3 - не верны.

Задача 1

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello Job4j!");
    }
}

Задача 2

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello");
        System.out.println("Job4j!");
    }
}
