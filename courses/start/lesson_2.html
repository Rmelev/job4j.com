Урок №2. Установка инструментов.


В этом уроке установим необходимые инструменты для прохождения курса.

Инструменты:
Комплект разработчика JDK
Среда разработки IntelliJ IDEA

Тест 1


Что не входит в JDK:

  1. Компилятор Java
  2. Документация
  3. JRE
  4. Интегрированная среда разработки
  5. Стандартные библиотеки классов

Тест 2


Кто разработал язык Java:

  1. Компания Google
  2. Компания Oracle
  3. Компания Sun Microsystems
  4. Компания Microsoft

Задача 1


В среде разработки IntelliJ Idea зайдите в:
File -> Settings -> Appearance & Behavior -> Appearance.
В поле Theme выберите самый верхний вариант из списка и нажмите Apply. Что при этом произойдёт?

Задача 2


В среде разработки IntelliJ Idea в левой панели дерева проекта раскройте текущий проект, после этого нажав правой кнопкой мыши на папке src создайте Java класс HelloWorld.

Ответы:


Тест 1

4 - интегрированная среда разработки не входит в комплект JDK.


Тест 2

3 - Java была разработана компанией Sun Microsystems, которая впоследствии была приобретена компанией Oracle
