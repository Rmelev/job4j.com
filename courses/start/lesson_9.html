Урок №9. Логические операторы.


В этом уроке мы рассмотрим логические операторы И, ИЛИ, НЕ - как они обозначаются, для чего используются, а также разберём примеры по каждому оператору.

Тест 1


Напишите результат каждого выражения, если у вас возникнут трудности, выведите на печать

  1. true || false
  2. true && false
  3. false || false || !true
  4. true && true || false

Тест 2


Оператор И (&&) является более приоритетным чем ИЛИ (||), но выражение в скобках имеет наивысший приоритет (выше чем &&). В выражении true || true && false || false расставьте скобки так, чтобы выражение в итоге приняло значение false:

  1. true || (true && false) || false
  2. true || (true && false || false)
  3. (true || true) && (false || false)
  4. (true || true && false) || false

Задача 1


Написать метод check c 2 boolean аргументами, который будет возвращать true только в случае если первое значение true, а второе false, во всех остальных вариантах метод должен возвращать false.

Задача 2


Написать метод checkThree c 3-мя boolean аргументами, который будет возвращать true только если все значения true или все значения false, во всех остальных случаях он должен возвращать false.

Ответы:

Тест 1

1. true 2. false
3. false 4. true


Тест 2

3. (true || true) && (false || false) - примет значение false. Сначала вычисляются выражения в скобках и получается true && false, что в итоге даёт false.

Задача 1

public boolean check(boolean a, boolean b) {
    return a && !b;
}

Задача 2

public boolean checkThree(boolean a, boolean b, boolean c) {
    return a && b && c || !a && !b && !c;
}

Комментарий к задаче 2: когда а И b И c равны true ИЛИ когда ни a и ни b и ни с не равны true - значение будет принимать true.
