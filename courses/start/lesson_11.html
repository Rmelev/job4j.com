Урок №11. Циклы.


В этом уроке мы разберёмся что такое цикл, какие разновидности циклов используются в языке Java, а также более подробно остановимся на двух из них: for и while.

Тест 1


Определите неверную конструкцию for или while

  1. for (int i = 0; i >= 0; i++) {...}
  2. for (int i = 0; i < 10; i++) {...}
  3. while (a > b) {...}
  4. while () {...}

Тест 2


Не используя программных средств, посчитайте, сколько раз в консоль будет выведено сообщение test: 

for (int i = 8; i > 2 ; i -= 2) {
     for (int j = 0; j <= 2; j++) {
         System.out.println("test");
     }
}

Задача 1


Написать метод evenRange, который принимает в качестве аргументов 2 числа и выводит все чётные числа в диапазоне от 1-го до 2-го числа.

Задача 2


Написать класс MultipleTable с методом show, который принимает одно число и выводит в консоль таблицу умножения в таком виде (например для числа 9): 

1 2 3 4 5 6 7 8 9
2 4 6 8 10 12 14 16 18
3 6 9 12 15 18 21 24 27
4 8 12 16 20 24 28 32 36
5 10 15 20 25 30 35 40 45
6 12 18 24 30 36 42 48 54
7 14 21 28 35 42 49 56 63
8 16 24 32 40 48 56 64 72
9 18 27 36 45 54 63 72 81

Ответы:

Тест 1

Вариант 1 можно написать, но цикл будет выполняться бесконечно
Вариант 4 не верен, не указано логическое выражение в скобках.

Тест 2

Внешний цикл будет выполняться 3 раза, вложенный цикл будет выполняться 3 раза, итого сообщение будет выведено 3 * 3 = 9 раз.

Задача 1

public void evenRange(int a, int b) {
    for (int i = a; i <= b; i++) {
        if (i % 2 == 0) {
           System.out.println(i);
        }
    }
}

Задача 2

public class MultipleTable {
    public void show(int a) {
        for (int i = 1; i <= a; i++) {
            for (int j = 1; j <= a; j++) {
                System.out.print(i*j + " ");
            }
            System.out.println();
        }
    }

    public static void main(String[] args) {
        MultipleTable mt = new MultipleTable();
        mt.show(9);
    }
}
