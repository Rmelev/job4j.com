Урок №4. Переменные.


В этом уроке вы познакомитесь с термином переменная, какие базовые операции можно проводить с переменными, правила наименования переменных и т.д.

Тест 1


Где неправильно объявлена переменная:

  1. result = 2;
  2. int money = 100;
  3. int x;
  4. int result;
    int result = 3;
  5. int mass;
    mass = 3;

Тест 2


Какие варианты названия переменных верны:

  1. Cat
  2. dog
  3. massOfCat
  4. ColorOfCat
  5. catNAME

Задача 1

Объявить переменные типа int: a, b, c и d. После этого присвоить им значения в следующем порядке.

  1. a присвоить значение 3
  2. c присвоить значение 2
  3. b присвоить значение c
  4. d присвоить значение a
  5. a присвоить значение b
  6. c присвоить значение d
Вывести на печать a, b, c и d. Проанализировать результат.

Задача 2

Объявить 4 переменных int: a, b, c, d.
Присвоить им произвольные значения.
Нужно сложить значения всех переменных и вычесть из суммы 10.
Результат вывести на печать в консоль.

Ответы на тесты:

Тест 1

Вариант 1 не верен - не указан тип переменной.
Вариант 4 не верен - повторно объявлена переменнуя с таким же именем.

Тест 2

Правильный ответ 2 и 3. Название переменной должно начинаться с маленькой буквы, каждое следующее слово в названии переменной с большой буквой, а остальные все буквы маленькие.

Ответы на задачи:

Задача 1

a = 2
b = 2
c = 3
d = 3

Задача 2

int result = a + b + c + d - 10;
System.out.println(result);
Результат для чисел 0, 1, 2, 3 будет равен -4.
