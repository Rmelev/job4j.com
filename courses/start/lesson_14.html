Урок №14. Правила оформления кода.


В этом уроке мы рассмотрим основные правила правила оформления кода, в том числе JavaDoc, правила наименования, что такое MagicNumbers и т.д..

Тест 1


Какие дескрипторы JavaDoc не существуют:

  1. @author
  2. @param
  3. @method
  4. @return
  5. @Test

Тест 2


Что такое Magic Numbers -

  1. Все числа которые встречаются в коде;
  2. Очень большие или очень маленькие числа;
  3. Произвольные числа, которые встречаются в коде без описания и их смысл не очевиден.

Задача 1


Найдите ошибки оформления в приведённом коде: 

public String CheckSettings(int a, int b) {
    if (a == 1920 && b == 1080) {
        return "1080p";
    } else if (a == 1280 && b == 720) {
        return "720p";
    } else {
        return "Unsupported resolution";
    }
}

Задача 2


После нахождения всех ошибок в задаче 1 - напишите для этого метода JavaDoc.
